# FloatPicture（悬浮图片）
----------  
安卓图片悬浮窗软件（Floating Image App for Android）

使用本软件可以在手机屏幕上以悬浮窗的形式显示任意自定义的图片。

***图片格式支持（Picture format）：jpg/jpeg、png、webp、bmp、gif（将作为静态图）、HEIC/HEIF。

***语言支持（Language）:  简体中文与英语（Simplified Chinese and English）


# 应用功能(Application Function)
----------  

###  1.设定图片大小(Resize the picture)
可根据手机屏幕大小自由调整照片的长宽（缩放比率精确到0.001），支持滑块拖动和数字输入，也可锁定原始比率调整。

###  2.设置图片显示角度(Set picture show degree)
可通过拖动滑块以45°为单位快速步进调整，也可手动输入精确的数值调整（最小单位精确到1°）。

注：如果旋转不生效请确保勾选“允许悬浮窗超出屏幕”。

###  3.设置照片透明度(Set picture transparency)
范围从0-1(最小单位精确到0.01)。

### 4.设置窗口位置(Set window position)
可设置悬浮图片在屏幕上的位置，支持滑块拖动和数字输入。

### 5.触摸并移动窗口(Touch and move the window)
勾选后仅可在“图片设置”界面中随意拖动图片的位置。（勾选后可能因为图片过大导致挡住其他按钮，无法触碰其他按钮，暂时不建议使用此功能）

*注：在首页侧边栏中的“全局设置”→“高级功能”→“可触摸的位置编辑”勾选后，可以在任意界面拖动（包括后台运行本软件并打开其他应用时），**但仍然由于图片挡住画面导致可能无法点击其他程序的按钮，所以也不推荐使用此功能**。

### 6.允许悬浮窗超出屏幕(Allow picture over layout)
勾选后可允许悬浮图片超出屏幕范围：

**可支持将图片尺寸调整到原始照片的1.1倍；

**可支持超出屏幕范围的旋转；

**可支持修改图片位置到屏幕尺寸2倍以内的距离。

### 7.可设置导入时的图片质量(Picture quality)

在首页侧边栏“全局设置”→“通用设置”→“图片质量”，需要在导入图片之前设置好参数，程序初始设置为80%。
